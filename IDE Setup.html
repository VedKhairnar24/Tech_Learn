Step-by-Step IDE Setup (VS Code)

1. Install VS Code

  • Go to the official VS Code website.
  • Download the version for your operating system (Windows, macOS, or Linux).
  • Install VS Code by following the installation wizard.
Tip: Make sure to check the system requirements on the website before downloading!

2. Install Extensions

VS Code extensions enhance functionality for specific languages and tools. Here's how to install them:

  1. Open VS Code.
  2. Go to the Extensions Marketplace:
    • Click the Extensions icon in the sidebar (or press Ctrl+Shift+X).
  3. Search for and install the following extensions based on your programming language:
    • For Python
      • Python (by Microsoft): Adds Python language support.
      • Pylance: Provides fast type-checking and IntelliSense.
      • Black Formatter: Automatically formats Python code.
    • For JavaScript/Web Development
      • Live Server: Launches a local development server with live reload.
      • Prettier: Automatically formats JavaScript, HTML, and CSS.
      • ESLint: Lints JavaScript code for errors and style issues.
    • For Java
      • Extension Pack for Java (by Microsoft): Includes Java language support, debugging, and Maven/Gradle tools.
    • General Extensions
      • GitLens: Enhances Git functionality.
      • Code Runner: Runs code snippets directly in VS Code.

3. Set Up for Specific Languages

Python Setup

To set up Python, follow these steps:

  1. Install Python:
    • Download Python from python.org.

    • During installation, check the box to Add Python to PATH.

    • Verify installation: Open a terminal and run the following command: 
      python --version


  2. Create a Virtual Environment:
    • Open a terminal in VS Code (Ctrl+`).

    • Navigate to your project folder: 
      cd path/to/your/project

    • Create a virtual environment: 
      python -m venv venv

    • Activate the virtual environment:
      • Windows: 
        venv\Scripts\activate

      • macOS/Linux: 
        source venv/bin/activate



  3. Install Packages: 
    pip install numpy pandas


  4. Run Python Code:

    Create a file hello.py:

    print("Hello, World!")

    Run it:

    python hello.py

Tip: Make sure to activate your virtual environment before installing packages!
JavaScript/Web Development Setup

To set up JavaScript, follow these steps:

  1. Install Node.js:
    • Download Node.js from nodejs.org.

    • Verify installation: Run the following commands: 
      node --version

      npm --version


  2. Create a Project:

    Open a terminal in VS Code and initialize a new Node.js project:

    npm init -y


  3. Install Packages: 
    npm install express


  4. Run JavaScript Code:

    Create a file index.js:

    console.log("Hello, World!");

    Run it:

    node index.js

Note: Ensure you have Node.js installed before creating a project!
Java Setup

To set up Java, follow these steps:

  1. Install Java Development Kit (JDK):
    • Download the JDK from Oracle or use OpenJDK.

    • Verify installation: Run the following commands: 
      java --version

      javac --version


  2. Set Up Java in VS Code:
    • Install the Extension Pack for Java (as mentioned above).

    • Open a Java file (.java), and VS Code will prompt you to configure the Java Runtime.


  3. Run Java Code:

    Create a file HelloWorld.java:

    public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

    Run it by clicking the Run button in VS Code.

Reminder: Ensure you have the JDK installed before running Java code!

4. Configure Settings (Optional)

  • Format on Save:
    • Go to File > Preferences > Settings (or press Ctrl+,).
    • Search for "Format On Save" and enable it.
  • Change Theme:
    • Go to the Extensions Marketplace and search for themes like One Dark Pro or Material Theme.

5. Debugging

  • Set Breakpoints: Click to the left of the line numbers to set breakpoints.
  • Run Debugger: Press F5 to start debugging.
  • Use the Debug sidebar to inspect variables and step through code.

6. Version Control with Git

  1. Install Git:
    • Download Git from git-scm.com.
    • Verify installation: Run git --version.
  2. Initialize a Git Repository:
    • Open a terminal in your project folder: 
      git init
  3. Commit Changes:
    • Stage files: 
      git add .
    • Commit changes: 
      git commit -m "Initial commit"
  4. Connect to GitHub:
    • Create a repository on GitHub.
    • Link your local repository: 
      git remote add origin https://github.com/your-username/your-repo.git
  5. Push changes: 
    git push -u origin main
